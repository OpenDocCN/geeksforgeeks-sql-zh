函数是一种特殊类型的预定义命令集，它执行一些操作并返回一个值。函数对提供给它们的零个、一个、两个或多个值进行操作。提供给函数的值称为参数或自变量。

MySQL 函数被分为不同的类别，如字符串函数、数学函数、日期和时间函数等。虽然 MySQL 提供了大量的函数，但是在本讨论中，我们将只讨论一些个人函数。
